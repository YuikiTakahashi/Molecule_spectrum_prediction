# Molecule_spectrum_prediction
Predict the spectrum of molecules like YbOH

## Installation

### Using pip (recommended)
```bash
pip install -r requirements.txt
```

### Using conda
```bash
# Install PyTorch first (for CPU or GPU)
# For CPU:
conda install pytorch cpuonly -c pytorch

# For GPU (CUDA):
conda install pytorch cudatoolkit -c pytorch

# Then install other dependencies
pip install -r requirements.txt
```

### Manual installation
If you prefer to install packages individually:
```bash
pip install numpy pandas sympy torch ujson scipy matplotlib seaborn ipython scikit-learn jupyter
```

## Dependencies
- numpy
- pandas
- sympy
- torch (PyTorch)
- ujson
- scipy
- matplotlib
- seaborn
- ipython
- scikit-learn
- jupyter
